# choochoo_arduino_term
Arduino Game Project

# Overview
- Board: Arduino Mega 2560
- Module
    - 2.4" TFT LCD
    - HC-06(bluetooth)
    - 7 Segment
    - Potentmeter
    - 4-axis joystick
    - Buzzer
    - LED(R,G,Y)
- Games
    - Maze
    - Touch Circle
    - Find Pitch of Sound
    - Select(Cut) the LED
